Descripción del Proyecto

Este proyecto reúne cinco actividades desarrolladas en JavaScript con el objetivo de repasar y reforzar conceptos del lenguaje para luego pasar a aprender React.

Temas desarrollados

  • CRUD: Implementación de creación, lectura, actualización y eliminación de registros.
  • Callbacks: Uso de funciones como parámetros para manejar procesos.
  • Promesas: Manejo de procesos asíncronos en JavaScript.
  • Async / Await: Simplificación del trabajo con promesas.
  • Destructuring: Extracción eficiente de datos desde objetos y arreglos.

Tecnologías utilizadas

  • HTML5
  • Bootstrap 5
  • JavaScript
