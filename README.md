# Actividades JavaScript

Este proyecto reúne diferentes actividades desarrolladas en JavaScript con el objetivo de aplicar conceptos fundamentales y modernos del lenguaje.

##  Descripción

El trabajo contiene ejercicios prácticos enfocados en el manejo de datos, programación asíncrona y nuevas características de JavaScript.  

Cada actividad está organizada en módulos independientes para facilitar la comprensión y práctica de los temas vistos en clase.

---

##Tecnologías Utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

---

## Temas Desarrollados

### 1️ CRUD
Implementación de un sistema básico para:
- Crear registros
- Leer registros
- Actualizar información
- Eliminar datos

Se trabaja con manipulación del DOM y almacenamiento local.

---

### 2️ Callbacks
Uso de funciones como parámetros para manejar procesos y eventos.  
Se implementa una búsqueda simulada de países utilizando callbacks.

---

### 3️ Promesas
Manejo de procesos asíncronos utilizando `Promise`.  
Permite controlar resultados exitosos y errores en operaciones futuras.

---

### 4 Async / Await
Forma moderna de trabajar con promesas.  
Permite escribir código asíncrono de forma más clara y legible.

---

### 5 Destructuring
Uso de destructuración para extraer datos de objetos y arreglos de manera eficiente.

---

## Objetivo del Proyecto

Fortalecer la lógica de programación y comprender el funcionamiento de JavaScript tanto en procesos síncronos como asíncronos, aplicando buenas prácticas de desarrollo.
